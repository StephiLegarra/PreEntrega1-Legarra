
## Libraries used -->

- Material UI
- formik
- yup
- sweetalert2
- react spinners

## To use this project you will need the environment variables, create the .env file and assign the following to it

- REACT_APP_APIKEY=
- REACT_APP_AUTHDOMAIN=
- REACT_APP_PROJECTID=
- REACT_APP_STORAGEBUCKET=
- REACT_APP_MESSAGINGSENDERID=
- REACT_APP_APPID=
